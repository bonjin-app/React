# Opentutorials React

https://ko.reactjs.org/

사용자 정의 태그를 만들어서 사용한다.
이것을 Components 라고 한다.

## npm 설치
npm install -g create-react-app .
create-react-app -V

## npx
create-react-app 임시로 설치해서 딱 1번만 실행하고 지운다.
설치할 때마다 다운받기 때문에 항상 최신상태다.

## Start
npm run start

## Deploy
build 파일이 생긴다.

```bash
npm run build
```

## serve
build 파일을 서버를 통해 볼 수 있다.
npx serve -s build
